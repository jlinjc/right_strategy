【★最實用:看「📋 今日掛單建議」一塊就夠】
  dashboard 最上方那塊「📋 今日掛單建議」直接告訴你台美股『現在該怎麼下單』:
    🟢 可買(便宜)  = 站上200MA又沒追高、停損≤12% → 現在可進,掛限價別買貴於表上價。
    🟢 可買(偏貴小量)= 在趨勢中但離200MA遠 → 可買但系統自動把曝險調小,只小量。
    🟡 別追高      = 已超過進場門檻 → 別追!掛單等拉回到表上「掛單價」以下才成交。
    🔴 不要碰      = 跌破200MA 或 信用/費半哨示警 → 現金,等站回。
  每列給:動作 / 標的 / 掛單價(限價) / 停損 / 原因。照著掛單就好。
  ⚠️ 台股那段價位是「%」(yfinance絕對價會失真)→用你券商現價×那個%算限價。

【唯一要記的一件事】★只要雙擊「更新儀表板.bat」就好
  它會一次做完三件事:
    1. 更新所有資料(指數兩策略 + 個股信號 + 回測實證 + 選單)
    2. 自動啟動本機伺服器(會彈一個「小黑窗」)
    3. 自動用瀏覽器開好 http://localhost:8000/strategy_dashboard.html
  ※ 重要:那個「小黑窗」就是伺服器,看儀表板期間別關它。
    關掉小黑窗 = 儀表板會「資料載入失敗」。看完要收工再關。
  ※ 為什麼不能直接點 html:瀏覽器在 file:// 下會擋 fetch(=就是你看到的
    「資料載入失敗」)。一定要用伺服器以 http://localhost:8000 開,bat 已自動處理。

【已經開著、想看最新數字】
  先雙擊「更新儀表板.bat」更新,再回瀏覽器按 Ctrl+R / F5 即可。
  (想最快、只更新指數兩策略不更新個股 → 命令列跑 python update_dashboard.py --fast)

【平常的自動背景更新】
  排程會在這些時間自動更新資料(不彈視窗):
    早上 07:00(美股收盤定數)
    晚上 21:00 / 21:30 / 22:00 / 22:30 / 23:00 / 23:30(美股盤中)
  但「看」的時候仍要伺服器在跑:雙擊 更新儀表板.bat 開一次即可(它會重用已在跑的伺服器)。

【dashboard 怎麼看】★2026-06 改版:加「配置模式」一鍵 + B 內建30日鎖
  最上方「🎯 兩策略今日操作」：
    步驟1 右上角填一次「本金A / 本金B」(總額,會自動記住)。
    步驟2 ★點一個「配置模式」按鈕(三選一,會自動幫你算好 A/B 比例+槓桿)：
       🛡️ 純A·省事     → 全押SMH。最不費心、稅最省。CAGR+38.6%/回撤-35%。
       ⚖️ A+B各半·最穩  → A50/B50,不加槓桿。回撤最淺-26%、Sharpe最高1.45。
       🚀 A60/B40×1.35·財富 → 主力A六成+衛星B四成,整組×1.35槓桿。
                          同樣-34%回撤卻多賺+12%/年(CAGR+50.8%)=新發現的最佳財富點。
       (選了哪個,該按鈕會亮+打✓;想自己調比例就直接改本金A/B數字=自訂模式。)
    步驟3 看 🅰️A(抱SMH) / 🅱️B(輪動最強) 兩張卡,各自告訴你:
       動作、進場上限價、出場/停損價、買幾股≈多少$。
       - 動作=🟢持有可進 / 🟡追高等拉回 / 🔴出場現金 / 🟠恐慌容忍
       - 限價:別買貴於「進場上限價」；跌破「出場/停損價」→轉現金
       - 信用示警時兩策略都自動砍曝險到 0
  ★B 卡的「最低持有30日鎖」(這是新規則,你照著做就好)：
       - 卡片副標會顯示:現在抱哪檔、哪天進場、🔒鎖定中還剩幾天 / 🔓可換。
       - 🔒鎖定中 = 就算今天冒出更強的別動,抱滿一個月(≈21交易日)再換。
       - 🔓可換 = 已抱滿,若「今日最強」換人了,這次更新就會自動換過去。
       - 例外:B 抱的那檔若跌破200MA → 不管鎖沒鎖,立刻出場(系統自動處理)。
       - 你不用記日期,系統會從上次存的狀態自己算;你只要照卡片做。
  兩策略可同時用,各分一筆本金,互不干擾。
  個股(衛星):看下方信號卡,已含進場/停損/騎乘出場線/股數。
  ※ 看「收盤」訊號做決定;盤中數字是參考,別被洗。

【🇹🇼 台股版面板】★同一套引擎搬到台股(research_taiwan_core/optimize.py)
  位置:dashboard 上方美股面板下面那塊「🇹🇼 台股版今日操作」。
  核心宇宙:0050元大台灣50 / 0052富邦科技 / 006208富邦台50 / 0051中型100 / 00757統一FANG+。
    (排除高股息0056=均值回歸擇時會傷;主動式ETF 00991A/00981A 太新無法回測,只列觀察候選)
  🅰️ A 抱0052富邦科技(保守主力) / 🅱️ B 月輪動最強(衛星,內建最低持有30日鎖),看法同美股。
  信用哨:台股用「全球信用+費半 HYG+LQD+SOXX」(台股=半導體供應鏈,費半領先;本地匯率無效)。
  ⚠️ 台股看盤三件事(很重要):
    1. 一律看「%」:卡片給的是「距200MA、距50MA、進場上限+X%、停損-X%」=相對值,永遠正確。
       台股 yfinance「絕對價位」可能標錯(資料瑕疵),別照抄數字,用「%」×你券商的實際報價換算。
    2. 台股1張=1000股,ETF可買零股;下單前用券商現價套上面的%算進場/停損價。
    3. 看收盤訊號;盤中參考。
  【台股怎麼配置】★與美股相反:台股偏 B
    台股本地ETF「無資本利得稅」(只證交稅0.1%)→月輪動B的高換手沒有美股那種短期稅懲罰,
    淨報酬 B(+35.1%)反而贏 A(+32.1%)。→ 台股建議偏 B 月輪動,或 A+B各半。
    想衝財富:加 00631L 元大台灣50正2(內建2x免融資)當衛星,A40/B40+正2 20%
              =Sharpe2.21/回撤-11%。但正2須擇時(站上自己200MA才抱,崩盤會爆)、盤整有耗損,謹慎。
  熊市保護實證:2008金融海嘯 0052擇時 0% vs 買持 -48%(=+48%保護);2022 擇時+3.7% vs 買持-22%。

【兩策略怎麼選】★成本/稅後 + 核心衛星實證(research_core_satellite.py)
  A 全押SMH ：raw報酬高、稅最省、換倉少。
  B 輪動最強：Sharpe較高(1.45)、回撤淺(-26% vs A -35%)、抗半導體失寵。
  ★新發現(推翻舊「純A唯一解」):A+B 混合的 Sharpe(1.45)>純A(1.37),
    把混合加槓桿到「與純A同回撤(-34%)」,CAGR 反超純A:
      純A          → CAGR +38.6% / 回撤 -35%
      A60/B40 ×1.36 → CAGR +50.8% / 回撤 -34%(同回撤多賺+12%/年!)
  → 三種人三種選法:
     ‧ 不想加槓桿、求省事    → 純A
     ‧ 不想加槓桿、想更穩    → A+B 各半(回撤-26%、OOS最穩)
     ‧ 要最大財富、能扛-34% → A60/B40 整組×1.3~1.4(同回撤多賺一截,槓桿仍鎖1.5)

【輪動(B)操作手冊】★research_rotation_manual.py 掃過頻率/遲滯/最低持有/稅緩衝
  ‧ 換倉頻率:月換是甜蜜點(週換稅太重、季換訊號鈍)。
  ‧ ★最低持有21日:買進後至少抱一個月再准換(跌破200MA仍立刻離場),
    全程+4個OOS窗 Sharpe/報酬/換手/稅後 全部不更差還小贏=免費降換手。
  ‧ 遲滯帶/稅務緩衝/鎖更久 → 都實測更差,別用。乾淨的月換+最低21日最佳。

【槓桿怎麼設】★Kelly實證(research_costs_kelly.py)
  RISK_MULT=1.0(平均曝險~135%)對數成長+28%/回撤-26%。
  Kelly數學說可加到2-4x,但那是「樣本裡沒真熊市」的幻覺,別信。
  紀律上限 1.0~1.5x;要更高=賭牛市續命,真熊市會重傷。
  保守睡得著 → 0.67;標準 → 1.0;積極(能扛-36%) → 最多1.5。

【槓桿/風險旋鈕】
  core_status.py 裡 RISK_MULT＝1.0（平衡，曝險最高1.5x）
  想保守→改 0.67；想博命→改 1.33。改完下次更新生效。
